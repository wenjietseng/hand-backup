# hand-backup
This repo contains a unity project with a runable package setup for oculus hand tracking


## specs and packages
- Oculus Quest: version 20.0.0.431.509.242741616
- Oculus Utilities v1.47.0
- OVRPlugin v1.48.0
- SDK v1.53.0.
- Unity v2019.4.3f1
